# recover
> Recover a recording that failed to upload

Use this command to attempt to recover a recording that failed to upload

## Syntax
```
/recover <session id>
```

<details>
  <summary>Example</summary>

  ```
  /recover 01GWHZZESP046N8TR90RFMEHYP
  ```
</details>
