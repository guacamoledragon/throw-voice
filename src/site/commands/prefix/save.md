# save
> Saves the current recording and uploads it to current, `saveLocation`, or specified text channel. _(Max recording is 256MB)_

!> **Deprecated:** Prefix commands will be removed on May 1, 2026. Please use [slash commands](/commands.md#slash-commands) instead.

```
!save [text-channel]
```
<details>
  <summary>Example</summary>

  ```
  !save
  !save bot-testing
  ```
</details>
