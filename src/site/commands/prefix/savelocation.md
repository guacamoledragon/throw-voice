# saveLocation
> Default text channel for all messages. Use `off` to restore default behaviour.

!> **Deprecated:** Prefix commands will be removed on May 1, 2026. Please use [slash commands](/commands.md#slash-commands) instead.

```
!saveLocation [text-channel | off]
```
<details>
  <summary>Example</summary>

  ```
  !saveLocation
  !saveLocation bot-testing
  !saveLocation off
  ```
</details>
